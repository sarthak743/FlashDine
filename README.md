# 🍽️ FlashDine

FlashDine is a full-stack food ordering web application built to deliver a fast, seamless, and intuitive user experience. It allows users to browse food items, add them to a cart, and place orders efficiently.

---

## 🚀 Features

- Browse food items with details  
- Add to cart and manage orders  
- User authentication (Login / Signup)  
- Place and track orders  
- Responsive UI  
- Fast performance using Vite  

---

## 🏗️ Tech Stack

### Frontend
- React.js  
- Vite  
- Tailwind CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/your-username/FlashDine.git  
cd FlashDine  

---

### 2. Setup Frontend

cd src  
npm install  
npm run dev  

---

### 3. Setup Backend

cd server  
npm install  
npm start  

---

## 🔧 Environment Variables

Create a `.env` file inside the server folder and add:

PORT=5000  
MONGO_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key  

---

## 🧪 Scripts

### Frontend

npm run dev      # Start development server  
npm run build    # Build for production  

### Backend

npm start        # Start backend server  
npm run dev      # (if using nodemon)
